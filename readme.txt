This program can generate encryption keys for RSA435, and use them to
sign and verify documents

to compile, add all other files listed here to the project directory that
contains the provided project files, replacing the existing rsa435.cc and the
provided makefile as well.

Program can be run with s and v arguments as well as the filename
Example: ./rsa435 s file.txt
Additionally, adding a k argument instead will generate keys.
